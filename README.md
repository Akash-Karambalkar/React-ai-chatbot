🚀 Built a Google Gemini-style AI Assistant using React

💡 Key Features:
✅ Ask anything and get AI-generated responses
✅ Clean chat interface with smooth UX
✅ Stores previous queries using localStorage
✅ Click on history to re-fetch responses instantly
✅ Loading states for better user experience
✅ Option to clear chat history

🛠️ Tech Stack:
React.js | Tailwind CSS | JavaScript | REST API Integration

📌 What I learned:

- Handling async API calls efficiently
- Managing state and UI updates in React
- Improving user experience with loading states and history features
- Building a responsive and clean UI
